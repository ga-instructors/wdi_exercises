#String Manipulation Templates

###Overview:
* In Ruby, write a method that takes in a given string and returns the specified output.

###Spec:
* Given: 'string'
	* Return: `'string'
* Given: '$^-^$'`
	* Return: '-'
* Given: '$^h^$'
	* Return: 'H'
* Given: '$^Happy Monday^$'
	* Return: 'HAPPY MONDAY'
* Given: '$^Happy Monday^$ to you. $^right^$'
	* Return: 'HAPPY MONDAY to you. RIGHT'
